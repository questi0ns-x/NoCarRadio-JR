# NoCarRadio-JR

Desactiva la radio del vehículo al subirte a cualquier coche en FiveM.

## Características

- Desactiva completamente la radio al entrar en un vehículo
- Funciona con todos los tipos de vehículos
- Hilo de respaldo para asegurar que la radio se mantenga desactivada

## Instalación

1. Copia la carpeta `NoCarRadio-JR` en tu carpeta `resources/[scripts]/`
2. Añade `ensure NoCarRadio-JR` a tu `server.cfg`
3. Reinicia el servidor

## Requisitos

- FiveM Server
- ESX Legacy (opcional - funciona standalone)

# NoCarRadio-JR (English)

Disable car radio when entering any vehicle in FiveM.

## Features

- Completely disables vehicle radio on enter
- Works with all vehicle types
- Backup thread ensures radio stays disabled

## Installation

1. Drop `NoCarRadio-JR` folder into your `resources/[scripts]/` folder
2. Add `ensure NoCarRadio-JR` to your `server.cfg`
3. Restart server

## Requirements

- FiveM Server
- ESX Legacy (optional - works standalone)
